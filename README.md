# Kash | Landing Page

**Kash** connects local merchants with customers. Browse shops, compare prices, and save money.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
